# 霓虹跃迁 (Neon Transition)

一款霓虹风格的休闲反应类游戏，玩家需要在光球消失前完成点击以累积分数。

## 技术栈

- Phaser 3
- TypeScript
- Vite

## 游戏模式

### 🎮 关卡模式
- 6个精心设计的关卡，难度递进
- 每关有目标分数和时间限制
- 通关后解锁下一关

### 💀 生存模式
- 无尽挑战，生命耗尽即结束
- 多种类型光球（普通球、陷阱球、护盾球）
- 可选主动技能：时间冻结、全屏清除、紧急修复
- 狂热模式获得分数加成

## 成长系统

### 📋 每日悬赏
- 每天自动刷新 3 个短周期任务（简单/中等/困难各 1 个）
- 任务类型：得分、连击、点击数、游戏时长、生存时长、通关数量
- 完成后可领取金币奖励
- 进度跨局累积，次日自动重置
- 主菜单红点提醒待领取奖励

### 🏆 任务成就
- 12 个长期成就目标
- 涵盖得分、连击、关卡、点击数、游戏时长等维度
- 完成获得金币奖励

### 👤 玩家档案
- 等级与经验值系统
- 累计统计：得分、点击、连击、游戏时长等
- 关卡进度与最佳成绩
- 每日悬赏快速概览

## 运行方式

```powershell
npm install
npm run dev
```

打开 `http://127.0.0.1:6100`。

## 构建

```powershell
npm run build
```

产物输出到 `dist/` 目录。

## 项目结构

```
src/
├── config/          # 游戏配置（关卡、任务、每日悬赏）
├── managers/        # 核心管理器（游戏状态、存档存储）
├── scenes/          # 游戏场景（菜单、关卡、结算等）
├── types/           # TypeScript 类型定义
└── main.ts          # 入口文件
```

## 存档说明

游戏数据自动保存在浏览器 localStorage 中，包括：
- 玩家等级、经验、金币
- 关卡进度与最佳成绩
- 任务完成状态
- 每日悬赏进度（每日自动刷新）
